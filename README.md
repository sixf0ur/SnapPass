# SnapPass
A Xposed module to bypass snapchat screenshot detection.

## Supported Versions:
- 11.61.0.52
- 11.62.1.35
- 12.18.0.33

If the Snapchat version is compatible and the Hook loads you can see a Toast message when opening Snapchat.
